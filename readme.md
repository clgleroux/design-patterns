# The Magic Carptes (Design Patterns)

Create a Spaceship with 5 design patterns in Typescript [guideline](https://design-patterns-project.vercel.app/)

## Installation

Use node v.14

```bash
npm install
```

## Usage

Start

```bash
npm run start
```

Dev (with nodemon)

```bash
npm run dev
```

Test (with jest)

```bash
npm run test
```
